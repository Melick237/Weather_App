# Weather_App
As a user interested in the weather, I would like to find out about the weather forecasts for the next 5 days. 

- A city name can be entered via an input field.
- If no valid city name is entered, the forecast for "Berlin" is displayed.
- for the next 5 days is displayed
    o Date 
    o Min. temperature
    o Max. temperature
    o a description (e.g.: "Mostly cloudy")
     (if there are several descriptions for one day, the first one is displayed)
- all texts and formatting are in German
